# Halal Card Landing Page

Live Site → [https://halal-card.vercel.app/](https://halal-card.vercel.app/)

A modern, responsive landing page built for a card-providing platform.
This project focuses on clean layout structure, typography, and responsive design using pure HTML and CSS — no frameworks, no JavaScript.

---

## 🚀 Overview

Halal Card is a static marketing landing page designed to showcase different credit card options and features in a clean, structured layout.

The design emphasizes:

- Clear hierarchy
- Modern typography
- Responsive grid layouts
- Smooth UI interactions
- Professional fintech aesthetic

---

## 🛠 Tech Stack

- HTML5 (Semantic Markup)
- CSS3 (Flexbox & Grid)
- Google Fonts
  - Inter
  - DM Sans

- Fully Responsive Design
- No JavaScript
- No Frameworks

---

## 📱 Responsiveness

The website is fully mobile responsive.

Breakpoints are handled using CSS media queries to ensure:

- 3-column layout on desktop
- 2-column layout on tablet
- 1-column stacked layout on mobile

All elements scale properly across screen sizes.

---

## 🎨 Typography

Two fonts were used intentionally for hierarchy:

- DM Sans → Headings & hero sections
- Inter → Body text, supporting content, UI elements

This combination creates a clean fintech look with strong readability.
